# Admissions Operational Dashboard – Power BI

## Project Overview
This project analyzes student admission data using Microsoft Power BI.  
The dashboard provides insights into admission trends, student preferences, and scholarship eligibility.

## Dataset
The dataset contains 24,998 student admission records including:

- Student ID
- State
- Preferred Stream
- Entrance Exam
- Entrance Score
- Board Percentage
- Extracurricular Score
- Admission Probability
- Admission Status
- Scholarship Eligibility

## Key Questions Answered

1. How many students applied and how many were admitted?
2. Which academic streams are most preferred?
3. What is the gender distribution of applicants?
4. Which entrance exams are most common?
5. What percentage of students qualify for scholarships?
6. How do extracurricular scores influence admission probability?

## Key Insights

- Total Students: 24,998
- Admitted Students: 8,151
- Scholarship Eligible Students: 5,054
- Average Entrance Score: 77.89

Key observations:

- Management and Arts streams have the highest number of applicants.
- CET and NEET are the most common entrance exams.
- Approximately 20% of students qualify for scholarships.
- Higher extracurricular scores are associated with higher admission probability.
